# AI-Powered Sales Forecasting & Business Analytics Model

## 📌 Project Overview
This project builds a Machine Learning model to forecast monthly sales using historical Superstore data. It combines predictive analytics with interactive Power BI dashboards for executive decision-making.

---

## 🎯 Problem Statement
Businesses require accurate sales forecasts to:
- Optimize inventory
- Improve revenue planning
- Support financial forecasting
- Identify seasonality patterns

---

## 🔍 Feature Engineering
- Year & Month extraction
- Lag Features (Lag_1, Lag_2)
- Rolling Means (3-month & 6-month)
- Quarterly trends

---

## 🤖 Machine Learning Models
- Random Forest Regressor (Primary Model)
- Linear Regression (Baseline Comparison)
- Hyperparameter tuning with GridSearchCV

---

## 📊 Model Performance
- MAE: ~11024
- R² Score: ~0.65

---

## 🔮 Future Forecast
Generated 6-month forward forecast using walk-forward validation.

---

## 📈 Executive Dashboard (Power BI)
Includes:
- KPI Cards (Total, Average, Max Sales)
- Sales Trend Line with Forecast
- Rolling Mean vs Actual Comparison
- Lag Correlation Analysis
- Month-over-Month Growth %
