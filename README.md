🚀 AI-Powered Sales Forecasting & Business Analytics
📌 Project Overview

This project is an end-to-end AI-based Sales Forecasting System that predicts future monthly sales using Machine Learning and provides an interactive dashboard for business insights.

It combines:

Advanced Feature Engineering

Random Forest Regression

Performance Evaluation

Interactive Streamlit Deployment

Business Analytics via Power BI

🎯 Objective

To build a robust predictive model that:

Forecasts next month’s sales

Identifies trend patterns

Captures seasonality

Supports data-driven decision-making

📊 Dataset

Superstore Sales Dataset

Aggregated to monthly level

Features engineered from historical sales

⚙️ Feature Engineering

Lag Features (Lag_1, Lag_2, Lag_3)

Rolling Averages (3-Month, 6-Month)

Month & Year Index

Time-based trend encoding

These features help the model capture:

Trend

Momentum

Seasonality

Short-term fluctuations

🤖 Model Used
Random Forest Regressor

n_estimators = 500

max_depth optimized

Balanced bias-variance tradeoff

📈 Model Performance

MAE: 11,217.67

R² Score: 0.647

This indicates strong predictive performance for a real-world retail dataset.

🖥️ Streamlit Web Application

An interactive dashboard where users can:
